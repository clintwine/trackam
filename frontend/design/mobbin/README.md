# Mobbin References

Use this folder for screenshots and notes that influence the current auth, user-dashboard, and admin-dashboard surfaces.

## Suggested structure
- `auth/`
- `dashboard/`
- `admin/`
- `components/`

## Usage
- Name files descriptively.
- Add a short markdown note next to a screenshot when the intended takeaway is not obvious.
- Focus notes on hierarchy, density, states, and responsive behavior.
- Do not treat these references as a replacement for `frontend/docs/MOBBIN_DESIGN_WORKFLOW.md`, `frontend/intelligence.md`, or `docs/UI_ARCHITECTURE.md`.
